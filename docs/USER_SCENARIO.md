# 사용자 시나리오 (User Scenario) - 마이링크 (MyLink)

이 문서는 사용자가 마이링크 서비스에 접속하여 겪을 수 있는 '가능한 모든 시나리오'를 "사용자는 ___하기 위해 ___한다"의 형태로 작성한 문서입니다.

---

## 1. 방문자 (Visitor) 시나리오

### 1.1 프로필 접속 및 인지
* 방문자는 특정인의 여러 웹 링크를 한곳에서 확인하기 위해 소유자가 SNS 등에 공유한 고유 URL(`mylink.com/displayName`)을 클릭하여 접속한다.
* 방문자는 현재 접속한 페이지가 누구의 프로필인지 파악하기 위해 화면 최상단에 위치한 소유자의 실제 이름(username), 닉네임(displayName), 자기소개(Bio)를 확인한다.

### 1.2 링크 탐색
* 방문자는 자신이 관심 있는 콘텐츠나 사이트를 찾기 위해 프로필 아래로 나열된 링크 리스트를 스크롤한다.
* 방문자는 텍스트를 꼼꼼히 읽지 않고도 어떤 사이트인지 직관적으로 인지하기 위해 링크 좌측에 자동 출력된 아이콘(구글 파비콘)을 확인한다.

### 1.3 링크 클릭 및 이동 (클릭 카운트)
* 방문자는 원하는 목적지(포트폴리오, 인스타그램 등)로 이동하기 위해 해당 링크 버튼을 클릭한다.
* **(시스템)** 시스템은 추후 소유자에게 링크 선호도 통계를 제공하기 위해, 방문자가 링크를 클릭하는 순간 백그라운드에서 **해당 링크의 클릭 조회수를 1 증가**시킨다.
* 방문자는 원래 보던 마이링크 프로필 화면을 잃어버리지 않기 위해 새 탭(target="_blank")으로 열린 웹 브라우저에서 대상 사이트를 열람한다.

---

## 2. 소유자 (Owner) 시나리오

### 2.1 회원가입 및 초기 로그인
* 소유자는 마이링크 서비스에 가입하고 자신만의 프로필을 생성하기 위해 서비스 첫 화면에서 '구글 소셜 로그인' 버튼을 클릭한다.
* **(시스템)** 시스템은 소유자가 별도의 아이디를 짓는 수고를 덜어주기 위해 구글 지메일 아이디의 앞부분(예: `abc@gmail.com`의 `abc`)을 추출하여 초기 닉네임(displayName)이자 고유 URL 슬러그로 자동 설정한다.
* 소유자는 처음 가입하여 아무런 링크가 없는 상태에서 무엇을 해야 할지 알기 위해 화면 중앙에 표시된 "첫 번째 링크를 추가해보세요!" 안내 문구(Empty State)를 확인한다.

### 2.2 일반 로그인 및 로그아웃
* 소유자는 이전에 생성해 둔 자신의 링크들을 관리하기 위해 '구글 소셜 로그인'을 통해 다시 로그인한다.
* 소유자는 공용 기기에서 계정을 안전하게 보호하기 위해 관리를 마친 후 '로그아웃' 버튼을 클릭하여 서비스에서 빠져나온다.

### 2.3 프로필 정보 관리 (인라인 편집)
* 소유자는 방문자에게 보여줄 자신의 실명(username)을 변경하기 위해 이름 영역을 클릭해 **인라인 편집** 모드로 전환하여 텍스트를 수정하고 엔터(Enter)를 누른다.
* 소유자는 자신의 서비스 접속 고유 주소를 뜻하는 닉네임(displayName)을 변경하기 위해 해당 영역을 인라인 편집으로 수정한다. (수정 즉시 `mylink.com/새닉네임`으로 접속 주소가 변경됨)
* 소유자는 방문자에게 자신을 짧게 어필하기 위해 자기소개(Bio) 영역을 클릭해 인라인 편집으로 텍스트를 작성하거나 수정하고 저장한다.

### 2.4 새로운 링크 추가
* 소유자는 새로운 목적지 링크(예: 새로 올린 유튜브 영상)를 등록하기 위해 '새 링크 추가' 버튼을 클릭한다.
* 소유자는 별도의 팝업이나 페이지 이동 없이, 리스트에 새롭게 생성된 빈 입력 줄을 클릭하여 제목과 대상 URL을 인라인 편집으로 입력한다.
* **(시스템)** 시스템은 사용자의 입력 실수를 방지하기 위해 URL에 `http://` 또는 `https://`가 누락된 경우 이를 자동으로 추가하여 저장한다.
* 소유자는 링크 입력을 완료하고 즉시 서버에 반영하기 위해 엔터(Enter) 키를 누르거나 화면의 빈 곳을 클릭(onBlur)한다.
* 소유자는 자신이 입력한 URL이 정상적인 사이트인지 시각적으로 확인하기 위해 구글 파비콘 API를 통해 자동으로 렌더링된 좌측 아이콘을 확인한다.

### 2.5 기존 링크 수정
* 소유자는 예전에 등록한 링크의 제목에 오타가 있거나 목적지 URL이 변경되었을 때, 수정하려는 텍스트 영역을 직접 클릭하여 인라인 편집 모드로 진입한 뒤 내용을 고친다.
* 소유자는 수정한 내용을 저장하기 위해 엔터 키를 누르거나 바깥 배경을 클릭한다.

### 2.6 기존 링크 삭제
* 소유자는 더 이상 공유하고 싶지 않거나 유효기간이 끝난 링크를 리스트에서 완전히 제거하기 위해 해당 항목 옆의 '삭제' 버튼(휴지통 아이콘 등)을 클릭한다.
* 소유자는 삭제 여부를 재차 묻는 경고창 때문에 번거로움을 느끼지 않기 위해, **항목이 즉시 지워지는 것을 확인**한다.
* 소유자는 실수로 삭제 버튼을 눌렀을 경우 링크를 다시 복구하기 위해, 화면 하단에 뜬 **"삭제되었습니다. [실행 취소(Undo)]" 토스트 알림 팝업**을 클릭한다.

### 2.7 내 프로필 공유 (퍼블리싱)
* 소유자는 완성된 내 프로필을 다른 사람들에게 알리거나 SNS 프로필에 등록하기 위해 화면 상단의 '공유하기' 버튼을 클릭한다.
* 소유자는 URL을 드래그해서 복사하는 수고를 덜기 위해 클립보드에 내 고유 URL(`mylink.com/displayName`)이 자동 복사되었다는 팝업 알림을 확인한다.
